# CASB
## 什么是CASB?
CASB云访问安全代理 是介于企业用户和云服务提供商之间的安全策略执行点。CASB 可以结合多种不同的安全策略（从身份验证和凭据映射到加密、恶意软件检测等等），提供灵活的企业解决方案，帮助确保云应用在授权和未授权应用程序以及托管和非托管设备上的安全性。

## CASB的4大基础
- 可见性
- 数据安全
- 威胁防护
- 合规性
