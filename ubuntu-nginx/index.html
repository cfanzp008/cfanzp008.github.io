如何使用Ubuntu linux使用nginx部署简单的静态网站?

基本步骤

Ubuntu 使用linux部署简单的静态网页主要分为以下几步:

  1. 安装nginx
  2. 配置nginx
  3. 将网站放入nginx配置对应的网站目录
  4. 启动nginx
  5. 测试网站是否正常
