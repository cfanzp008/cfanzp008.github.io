# OpenScreen：免费开源的屏幕录制神器

## 什么是 OpenScreen

OpenScreen 是一款免费、开源的屏幕录制和演示视频制作工具，旨在帮助用户创建专业级别的产品演示视频。该项目在 GitHub 上已获得 **18,400+ 颗星**，是 Screen Studio 的优秀开源替代品。

### 核心理念

- **完全免费**：无需订阅，无广告，无水印
- **开源透明**：代码完全开放，可自由修改
- **商业可用**：MIT 许可证，允许商业使用

## 核心特性

### 1. 高质量屏幕录制

- 支持全屏录制
- 支持单个应用窗口录制
- 录制时显示光标追踪
- 高清视频输出

### 2. 智能缩放效果

- **自动缩放**：跟随鼠标移动自动缩放
- **手动缩放**：通过快捷键控制缩放
- **平滑过渡**：缩放效果流畅自然

### 3. 标注功能

- 添加文本注释
- 绘制箭头和形状
- 插入图片标注
- 高亮重要内容

### 4. 后期编辑

- 时间轴编辑
- 关键帧动画
- 裁剪和拼接
- 背景自定义

### 5. 导出选项

- 多种视频格式
- 自定义分辨率
- 质量调节
- GIF 导出

## 安装方法

### macOS

```bash
# 使用 Homebrew 安装
brew install --cask openscreen
```

或直接下载 DMG 文件从 GitHub Releases。

### Windows

从 GitHub Releases 下载 MSI 或 EXE 安装包。

### Linux

```bash
# 使用 AppImage
chmod +x OpenScreen-*.AppImage
./OpenScreen-*.AppImage

# 或使用 Flatpak
flatpak install flathub com.openscreen.OpenScreen
```

## 快速开始

### 1. 启动应用

安装完成后，打开 OpenScreen 应用。

### 2. 选择录制区域

- 全屏录制：选择整个屏幕
- 窗口录制：选择特定应用窗口
- 区域录制：自定义录制区域

### 3. 开始录制

点击录制按钮或使用快捷键开始录制。

### 4. 录制过程中的操作

- **缩放**：使用快捷键放大/缩小
- **标注**：添加注释和箭头
- **暂停**：暂停和继续录制

### 5. 导出视频

录制完成后，使用编辑器进行后期处理，然后导出为视频文件。

## 快捷键

| 操作 | 快捷键 |
|-----|--------|
| 开始/停止录制 | Cmd+Shift+2 |
| 暂停/继续 | Cmd+Shift+2 |
| 放大 | Cmd++ |
| 缩小 | Cmd+- |
| 重置缩放 | Cmd+0 |
| 添加标注 | Cmd+Shift+A |
