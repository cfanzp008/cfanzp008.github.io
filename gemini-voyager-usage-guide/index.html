Gemini Voyager - Google Gemini 全能增强插件

简介

Gemini Voyager 是为 Google Gemini 和 AI Studio 打造的全能增强插件，集成时间轴导航、文件夹管理、提示词库及聊天导出等众多功能。

GitHub: https://github.com/Nagi-ovo/gemini-voyager Stars: 15,800+ | Forks: 474+

支持浏览器

  • Chrome / Edge / Firefox / Safari / Opera / Brave
  • 基于 Chromium 的浏览器（Arc, Vivaldi 等）均支持

核心功能

📂 文件夹管理

  • 两层级文件夹 hierarchy
  • 拖拽排序支持
  • 支持 Account Isolation Mode
  • 自定义文件夹颜色

💡 提示词库（Prompt Vault）

  • 保存和复用提示词
  • 支持 Gemini、AI Studio 和自定义网站
  • 跨平台同步

☁️ 云同步

  • 文件夹和提示词同步到 Google Drive
  • 多设备无无缝使用

📍 时间轴导航（Gemini 专享）

  • 可视化节点跳转
  • 星级标记重要消息
  • 管理对话分支

💾 聊天导出

  • 导出为 JSON、Markdown 或 PDF
  • 包含图片

🧜‍♀️ Mermaid 渲染

  • 自动渲染流程图
  • 序列图和其他图表

🍌 NanoBanana

  • 无损去除 Gemini 生成图片的水印

🔬 Deep Research

  • 提取思考过程和研究链接

其他功能

  • Batch Delete: 批量删除对话
  • Quote Reply: 引用回复
  • Tab Title Sync: 浏览器标签同步
  • Prevent Auto Scroll: 阻止自动滚动
  • Input Collapse: 自动展开输入区域
  • Default Model: 设置默认模型
  • Hide Recent Items: 隐藏最近列表

  • Firefox Add-ons

# 生产构建
bun run build:chrome
bun run build:firefox
bun run build:all

使用技巧

文件夹管理

  1. 点击侧边栏的文件夹图标
  2. 创建新文件夹
  3. 拖拽对话到文件夹中
  4. 右键设置文件夹颜色

提示词库

  1. 打开扩展 popup
  2. 进入 Prompt Vault
  3. 点击保存当前对话为提示词
  4. 在新对话中快速调用

时间轴导航

  1. 点击时间轴图标
  2. 可视化查看所有消息
  3. 点击节点跳转到对应位置
  4. 星级标记重要消息

视觉特效

在设置中可以开启季节性特效：

  • 雪花效果
  • 樱花飘落
  • 电影雨

总结

Gemini Voyager 大幅提升了 Gemini 的使用体验，如果你经常使用 Gemini 对话，这款插件值得一试。

项目地址： https://github.com/Nagi-ovo/gemini-voyager 文档： https://voyager.nagi.fun/en
