# ubuntu如何安装打印机驱动？
## 这里以M7206w打印机为例
### 下载地址
- https://newsupport.lenovo.com.cn/driveList.html?fromsource=driveList&selname=M7206

### 安装方法：
```
sudo dpkg -i ./m7206wcupswrapper-3.0.1-1.i386.deb
sudo dpkg -i ./m7206wlpr-3.0.1-1.i386.deb
```
