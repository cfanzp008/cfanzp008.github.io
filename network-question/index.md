# 网络基础
## 什么事5元组？
- 源地址，源端口，目的地址，目的端口，协议。

## 网络五层协议是哪五层？
- 应用层:HTTP,SNMP,FTP,SMTP,DNS,Telnet
- 传输层:UDP,TCP
- 网络层:IP,ICMP,RIP,OSPF,BGP,IGMP,ARP,RARP
- 数据链路层:SLIP,CSLIP,PPP,FDDI,Arpanet,PDN,Ethernet
- 物理层:IEEE802.1A,IEEE802.2到IEEE 802.11
