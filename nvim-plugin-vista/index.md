# nvim插件Vista
Vista插件主要用来帮助显示tags，和Tagbar功能类似，在显示markdown文档结构的时候非常友好。推荐使用。
## 安装依赖
- ctags有2个版本，这里注意使用universal-ctags
```bash
exuberant-ctags
# install
#sudo apt install exuberant-ctags
# remove
sudo apt remove exuberant-ctags

sudo apt install universal-ctags
```
- 查看版本
