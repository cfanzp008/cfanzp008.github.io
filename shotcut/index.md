# 视频剪辑软件shotcut基本用法

## 安装
### mac安装
- 官网地址：https://www.shotcut.org/
- M1下载arm64的包
```shell
wget https://www.fosshub.com/Shotcut.html?dwl=shotcut-macos-ARM64-221221.dmg
```

## 使用注意事项
- 项目面板展示的才是剪辑的视频，源展示的是素材资源的视频。
