## 包过滤
- 查看源地址为192.168.0.2的包
```
ip.src_host == 192.168.0.2
```
- 查看源地址为192.168.0.2,目的地址为192.168.0.3的包

```
ip.src_host == 192.168.0.2 and ip.src_host == 192.168.0.3
```

- 查看源地址或目的地址为192.168.0.2的包
```
ip.addr == 192.168.0.2
```
